POST /v1/disk/resources/copy
1. Скопированный файл идентичен оригиналу.
2. Происходит или не происходит копирование в несуществующую папку(создается ли она сама)
3. Флаг overwrite работает исправно.
4. Проверка асинхронного копирования
5. При копировании самого в себя ничего не происходит(должна возвращаться ошибка)
6. При копировании несуществующего файла должна отдаваться ошибка

GET /v1/disk/resources/download
1. Файл можно скачать по полученной ссылке
2. При изменении id ссылки не должен появиться доступ к другим файлам
3. При многократном вызове ссылка должна либо перезатиратся либо отдаваться одна и та же
4. Ошибка при несуществующем файле

GET /v1/disk/resources/files
1. Список упорядочен
2. Присутствуют все имеющиеся файлы в отдаче
3. Работают фильтры
4. Лимит на вывод

POST /v1/disk/resources/move
1. Перемещенный файл идентичен оригиналу.
2. Происходит или не происходит перемещенние в несуществующую папку(создается ли она сама)
3. Флаг overwrite работает исправно.
4. Проверка асинхронного перемещенния
5. При перемещеннии самого в себя ничего не происходит(должна возвращаться ошибка)
6. При перемещеннии несуществующего файла должна отдаваться ошибка
7. При перемещеннии удаляется оригинал

GET /v1/disk/resources/public
PUT /v1/disk/resources/publish
PUT /v1/disk/resources/unpublish
1. После публикации файл появляется в выдаче списка опубликованных
2. При снятии с публикации пропадает из списка
3. При скачке скачивается именно тот файл
4. После снятия с публикации ссылка становится не валидна
